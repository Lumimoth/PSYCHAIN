Game Info

• Controls:

There are various points through the game that have diffrent controls. 3D sections use WASD for movement with the Mouse to look around. For the rest of the game use controls that would reflect that of a computer. You can grab tabs and move tabs around by clicking on the tab bar. Double click to open up an application. (basically just regurlar old computer controls)


• First Boot:

Please note, the game may take a short while to start on the first boot of the game. This is because it's creating and generating various files needed for the game to run.


• Window:

The game will boot up onto the primary monitor in full screen, to change primary monitor it is best to change it in the system display settings

The game is designed for 16:9 with 100% screen scale so if you are finding issues in other ratios seek the "windowed" option in the settings *note it is NOT recormended to play in windowed mode use as last resort.


• Saving:

The game will save automatically at set points during the story.


• Close / Pause the game:

You can press "escape" key to open the pause button.

Additionally the game can be closed through "steam force close" or by using the "Alt+F4" shorcut

Technical Warnings

Contains spoilers: (!) you can hover over the spoilers for places that require more clarification or contain spoilers.

This game (Phantomware 0) will interact with your computer's settings and files in unconventional ways. Below are the effects the game will have on your computer:



• Desktop Wallpaper

Phantomware 0 will modify your desktop wallpaper, creating custom images during gameplay.
These images are saved in the game's install directory at: Phantomware 0\Phantomware 0_Data\StreamingAssets

​The game will read your desktop wallpaper from your windows registry, It will then make a couple versions of it with various filters. In game, there will be an event where it changes your desktop wallpaper, and after you go back to the game, it will revert it.


• Registry Modifications

Phantomware 0 will access, store, and modify entries in the Windows Registry.

This is to change the desktop wallpaper. (Basically this allows the game to temporarily change desktop wallpaper without any permanent effects).


• File Access

By running Phantomware 0, you grant the game access, storage, and modification to data on all drives of your computer.

The game will access the file to your desktop wallpaper, and create, modify and store data in the games folder Phantomware 0\Phantomware 0_Data\.


• Application Interaction

Phantomware 0 may interact with open applications on your computer, including modifying, opening, or closing them, example: browser.


• Recovery Mode

All effects caused by Phantomware 0 are automatically reverted when the game is properly closed. In the event of a crash or sudden interruption, the game will attempt to recover and restore your computer's state to its original settings.


• Recording

This game can be streamed; however, it is recommended to record this experience so unwanted information can be appropriately censored. If you want to stream this game here are some things in particular you might want to know about… [CONTAINS SPOILERS]

Phantomware 0 contains an immersive scare that directly interacts with your desktop.

At a certain point in the game, Phantomware 0 will minimize itself and reveal your actual desktop, replacing your wallpaper with a custom horror image as part of a scripted jumpscare. During this moment, anything on your desktop — including icons, files, and shortcuts — will be visible.

If you do not want viewers to see what’s on your desktop, please take time beforehand to hide or clean up any sensitive or personal files, folders, or icons. This includes removing shortcuts, documents, or any content you wouldn’t want appearing on stream or in a recording.

As part of the scare, the game will also generate a temporary “README.txt” file on your desktop. Both the wallpaper and the file will automatically revert/disappear once you return to the game by maximizing the window.

This feature is purely cosmetic and non-malicious, intended to enhance the psychological horror experience.

✅ No personal data is collected or uploaded.
✅ No webcam, microphone, or folder access is used.
✅ Only your wallpaper is temporarily changed, and one file is briefly placed on the desktop.
